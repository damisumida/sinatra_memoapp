# memoapp

メモを追加編集削除できるアプリケーションです。

# 利用方法
## インストール方法
1. 緑色の `Code` ボタンを押してください。
2. `Download ZIP`を押し、任意のディレクトリに保存してください。

## 起動方法
1. 保存したディレクトリに移動してください。

```
$ cd directry_pass
```
2. メモアプリを起動してください。

```
$ bundle execute memoapp.rb
```
3. ブラウザより、以下のURLにアクセスしてください。

```
http://localhost:4567
```

## 終了方法
1. アプリを終了する場合は、` Ctrl + C `を入力し、終了してください。
